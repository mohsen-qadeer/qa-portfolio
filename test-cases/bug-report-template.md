## Bug Report Structure

| Field              | Details                                      |
|--------------------|----------------------------------------------|
| Bug ID             | BUG-001                                      |
| Title              | Short, clear description of the bug          |
| Module             | Which module is affected                     |
| Sub-Module         | Which specific feature is affected           |
| Environment        | Production / Staging / Development           |
| Platform           | Web / Mobile (iOS / Android)                 |
| Browser/Device     | Chrome 120 / iPhone 14 / Samsung S23         |
| Severity           | Critical / High / Medium / Low               |
| Priority           | Critical / High / Medium / Low               |
| Status             | New / In Progress / Fixed / Closed           |
| Reported By        | Mohsen Qadeer                                |
| Reported Date      | DD/MM/YYYY                                   |
| Assigned To        | Developer name                               |

## Severity Guide

| Severity | Definition                                          |
|----------|-----------------------------------------------------|
| Critical | System crash, data loss, payment failure            |
| High     | Major feature broken, no workaround available       |
| Medium   | Feature partially broken, workaround exists         |
| Low      | Minor UI issue, cosmetic defect                     |

---

## Priority Guide

| Priority | Definition                                          |
|----------|-----------------------------------------------------|
| Critical | Must fix immediately, blocks release                |
| High     | Must fix in current sprint                          |
| Medium   | Fix in next sprint                                  |
| Low      | Fix when time allows                                |
